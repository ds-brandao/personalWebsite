# Welcome to My Blog

This is an example blog post included with the portfolio template. It demonstrates how the markdown blog system works.

## Getting Started

To add your own blog posts:

1. Create a new markdown file in `public/blog-posts/`
2. Add an entry to `public/config/articles.json` with the article metadata
3. Place your article image in `public/images/blog/`

## Markdown Features

This template supports full **GitHub Flavored Markdown** including:

- **Bold** and *italic* text
- `Inline code` snippets
- [Links](https://example.com)
- Images and media
